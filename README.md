Sniffy
======

Installation
------------
```sh
git clone https://github.com/blairbodnar/sniffy
cd sniffy
npm install -g ember-cli
npm install -g firebase-cli
npm install
bower install
ember server
```

Deployment
----------
```sh
ember build
firebase deploy
```
