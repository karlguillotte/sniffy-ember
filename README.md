# Sniffy

This README outlines the details of collaborating on this application.

## Installation
```sh
git clone https://github.com/blairbodnar/sniffy
cd sniffy
npm install -g ember-cli
npm install -g firebase-cli
npm install
bower install
```

## Running
```sh
ember server
```
Visit http://localhost:4200

## Building & Deployment
```sh
ember build
firebase deploy
```

## Running Tests

* `ember test`
* `ember test --server`
